Food Ordering Mobile Applicaton

Description

This is an food ordering mobile application made using android studio, coded using Java and uses SQLlite for its database.

Functions of the application

This application allows users to make orders by adding items to the cart from resturants.
This application allows users to create new user accounts and login to them.
Users have to be logged in to an account to complete the order.
The users are able to check thier order history.
Runs in both phone and tablet.
